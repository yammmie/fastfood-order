# fastfood-order

## 주제
패스트푸드 주문

<br>

## 목적
java에서 MySQL을 연결하는 방법을 익히고, 응용하기 위한 프로젝트

<br>

## 개발 환경
- java: jdk1.8.0_40
- MySQL: MySQL Server 5.0

<br>

## 맡은 역할
- 장바구니
  - DB와 연동하여 메뉴를 추가하거나 삭제할 때마다 CRUD 수행
- 주문하기
  - 구매 가능한 메뉴 목록 출력, 구매, 담기, 이동 기능
- 마이페이지
  - 지금까지 구매했던 주문 내역 목록 DB에서 가져와 출력
